# Serial Manager (Windows Desktop App)

This app lets you:
- Upload Excel/CSV of serial numbers (daily)
- Add a single serial manually
- Search any serial
- Prevent duplicates (case-insensitive) with pop-up alerts
- Store data locally in SQLite (serials.db)

## Quick Start (No coding needed)
1. Install Python for Windows (from python.org).
2. Clone or download this repo.
3. Double-click `build.bat` â†’ it will create `dist\SerialManager.exe`.

## Usage
- Upload Excel/CSV or add single serials.
- Duplicates are rejected with a popup.
- Search any serial number.
- Data stored locally in `serials.db`.
